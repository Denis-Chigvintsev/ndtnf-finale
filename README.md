Представлен бакэнд сервиса бронирования гостиниц.
Помимо этого представлено приложение для тестирования бакэнда.
Для запуска стягиваем себе на комп .  Env файл специально оставил , поэтому по идее все должно запуститься. Открываем файл yml. Далее последовательно запускаем службы: сначала mongo, потом бакенд, потом фронтенд. Приложение для тестирования находится по адресу localhost:3000.
Для того чтобы полноценно запуститься должен быть хотя бы один админ. чтобы его получить нужно зарегистрироваться по email secret_admin@yandex.ru паполь 999.
